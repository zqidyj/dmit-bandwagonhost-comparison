## FAQ

**Q：DMIT 和搬瓦工线路一样，为什么价格差这么多？**

A：表面上都是 CN2 GIA，但硬件代际不一样。DMIT 全系 AMD EPYC，搬瓦工部分老机房还在用 Intel Xeon E5。相同线路下，DMIT 的 CPU 和磁盘 IO 性能明显更好，但价格反而更低，这就是为什么老用户普遍认为 DMIT 性价比更高。

**Q：DMIT 的 Pro、EB、T1 系列怎么选？**

A：简单说：电信用户选 Pro，联通/移动用户选 EB，不需要国内优化或用来做落地中转选 T1。T1 最便宜，但对大陆用户体验不做保证，可能绕路很严重。

**Q：两家都有 CN2 GIA，搬瓦工的退款保证更长，是不是更值得信任？**

A：退款政策只是一个维度。搬瓦工 30 天退款确实更宽松，但 DMIT 的 3 天全额退款对于测试线路来说完全够用。至于信任度，两家运营时间都在 5 年以上，在中文 VPS 圈子里口碑都不差，没必要因为退款天数就完全否定其中一家。

**Q：DMIT 套餐经常缺货，买不到怎么办？**

**Q：用 DMIT 搭建的服务有没有被封 IP 的风险？**

A：有，但 DMIT 有专门的换 IP 政策——IP 被墙每 15 天可以免费换一次，自助操作不用等客服。这比很多商家处理起来方便很多。Premium/EB 系列都支持这个政策。

**Q：DMIT 新手注册下单有什么坑要注意？**
